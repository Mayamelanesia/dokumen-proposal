Proposal skripsi ini membahas tentang pengembangan Sistem Informasi Surat Menyurat berbasis web yang diterapkan pada Kantor Kelurahan Anotaurei Kecamatan Anotaurei.
Sistem ini dibuat untuk mengatasi permasalahan dalam proses administrasi surat yang masih dilakukan secara manual, seperti pengajuan surat, pencatatan data, hingga 
proses verifikasi yang memerlukan waktu cukup lama.
Melalui sistem yang dirancang, masyarakat dapat melakukan pengajuan surat secara online tanpa harus datang langsung ke kantor kelurahan. Di sisi lain, pihak admin 
kelurahan dapat mengelola data penduduk, memverifikasi pengajuan, serta mencetak dan mengarsipkan surat dengan lebih efisien dan terstruktur.
Dengan adanya sistem ini, diharapkan dapat meningkatkan kualitas pelayanan publik, mempercepat proses administrasi, serta meminimalisir kesalahan dalam pengolahan data surat.
